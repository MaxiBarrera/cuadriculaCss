Elementos secundarios (ítems)

Un contenedor de cuadrícula contiene elementos de cuadrícula .

De forma predeterminada, un contenedor tiene un elemento de cuadrícula para cada columna, en cada fila, pero puede diseñar los elementos de cuadrícula para que abarquen varias columnas y/o filas.

La propiedad de la columna de cuadrícula:

La propiedad grid-column define en qué columna(s) colocar un elemento.

Tú defines dónde comenzará el elemento y dónde terminará.
