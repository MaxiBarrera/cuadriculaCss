Elementos secundarios (ítems)

Un contenedor de cuadrícula contiene elementos de cuadrícula .

De forma predeterminada, un contenedor tiene un elemento de cuadrícula para cada columna, en cada fila, pero puede diseñar los elementos de cuadrícula para que abarquen varias columnas y/o filas.

La propiedad de la columna de cuadrícula:

La propiedad grid-column define en qué columna(s) colocar un elemento.

Tú defines dónde comenzará el elemento y dónde terminará.

Nota: La propiedad grid-column es una forma abreviada de propiedad para las propiedades grid-column-start y grid-column-end.

Para colocar un elemento, puede hacer referencia a los números de línea o utilizar la palabra clave "span" para definir cuántas columnas abarcará el elemento.


Ejemplo

Hacer que "item1" comience en la columna 1 y termine antes de la columna 5:

.item1 {
  grid-column: 1 / 5;
}
Check!

Ejemplo

Hacer que "item1" comience en la columna 1 y abarque 3 columnas:

.item1 {
  grid-column: 1 / span 3;
}
Check!

Ejemplo

Hacer que "item2" comience en la columna 2 y abarque 3 columnas:

.item2 {
  grid-column: 2 / span 3;
}
Check!
